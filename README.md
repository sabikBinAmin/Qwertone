# Qwertone 

> Turn any standard keyboard into a portable, customizable musical instrument.

Qwertone is an open-source Android project that transforms ordinary USB and Bluetooth keyboards into musical controllers.

Instead of requiring expensive musical instruments or MIDI controllers, Qwertone explores how much music can be created using the hardware many people already own.

---

## Why?

Many people want to learn, practice, or create music but don't own instruments such as:

- Piano
- Guitar
- Drum Set
- MIDI Keyboard

However, many already own a computer keyboard.

Qwertone exists to answer one simple question:

> **How much music can we create using only a standard keyboard?**

## Creator's Note

Qwertone started from a personal question.

I wanted to create and perform music, but I didn't own instruments like a piano, guitar, or drum kit.

What I did have was a mechanical keyboard.

Instead of buying new hardware, I decided to explore a different question:

**Can an ordinary keyboard become a creative musical tool?**

This project is my attempt to answer that question in public.

Every version, every bug, every improvement, and every song created with Qwertone will be part of that journey.

---

## Vision

Build an accessible music creation platform that allows anyone to:

- Practice music
- Perform casually
- Create original songs
- Experiment with different instruments

without purchasing dedicated music hardware.

## Philosophy

Qwertone is not trying to replace real musical instruments.

Instead, it aims to lower the barrier to music creation by making use of hardware people already own.
